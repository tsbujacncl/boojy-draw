## Design Considerations for Boojy Draw

Based on these examples, Boojy Draw should consider:

### Layout Philosophy
- **Krita-inspired:** Desktop-optimized with persistent panels for quick access
- **Procreate-inspired:** Clean, minimal UI with focus on canvas
- **Photoshop-inspired:** Professional tool organization

### Recommended Approach
1. **Left Panel:** Tool options (size, opacity, brush presets) - collapsible
2. **Center:** Large canvas area with maximum space
3. **Right Panel:** Layers and color picker - collapsible
4. **Bottom:** Status bar with zoom, canvas info, undo/redo shortcuts
5. **Top:** Menu bar + quick toolbar with most-used actions

### Key Decisions
- Make panels collapsible to maximize canvas space
- Support keyboard shortcuts for hiding UI (like Procreate's clean view)
- Keep tool options contextual (show only relevant settings)
- Use modern, clean design language (Material Design 3 for Flutter)
- Optimize for desktop (mouse + stylus) rather than touch-first

### Unique Features to Consider
- Quick tool switcher (Procreate-style)
- Customizable panel positions (Photoshop-style)
- Minimal mode for distraction-free painting (Procreate-style)
- Persistent left toolbar with most-used tools (Krita-style)
